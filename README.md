## 项目简介

本项目是一款纯语音控制的绘图工具。用户启动语音识别后，可以不使用鼠标或键盘，通过自然语言指令完成图形创建、样式控制、撤销重做、移动缩放和画布导出。

## 核心能力

- 语音指令识别：基于浏览器 Web Speech API 获取用户语音。
- 本地指令解析：支持中文和少量英文绘图指令，不依赖云端模型即可完成常用操作。
- 复杂指令拆解：支持“然后 / 再 / and / then”等连接词，把一句话拆成多步绘图动作。
- 绘图操作执行：支持矩形、圆形、线条、箭头、星星、文字。
- 容错反馈：无法理解时给出可执行示例，而不是静默失败。
- 作品导出：支持导出当前画布为 PNG。

## 已支持的语音指令示例

- `画一个红色实心圆形`
- `画一个蓝色矩形，然后添加一条绿色箭头`
- `画一个粗的紫色线条`
- `画一个黄色星星`
- `向右移动最后一个图形`
- `放大最后一个图形`
- `撤销`
- `重做`
- `清空画布`
- `导出`

## 运行方式

本项目无第三方运行依赖，使用浏览器即可运行。

```bash
npm start
```

然后访问：

```text
http://localhost:5173
```

建议使用 Chrome 或 Edge。麦克风和语音识别能力通常需要通过 `localhost` 或 HTTPS 访问。

## 测试方式

```bash
npm test
```

测试覆盖语音指令解析、复杂指令拆解、绘图状态、撤销重做和最后元素调整。
