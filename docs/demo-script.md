# Demo 视频脚本

目标时长：2-3 分钟。

## 1. 开场

大家好，这是七牛云 x XEngineer 暑期实训营第四批次题目二的参赛作品：AI 语音绘图工具。

这个工具的目标是让用户不使用鼠标和键盘，仅通过语音完成绘图创作。

## 2. 展示启动

打开页面后，点击“开始聆听”。浏览器请求麦克风权限时选择允许。

说明左侧是语音状态、识别文本和可用指令示例，右侧是绘图画布和指令记录。

## 3. 展示基础绘图

说：

```text
画一个红色实心圆形
```

说明系统识别语音后，本地解析出绘图动作，并在画布上添加圆形。

## 4. 展示复杂指令拆解

说：

```text
画一个蓝色矩形，然后添加一条绿色箭头
```

说明一句话被拆成两个动作，分别生成矩形和箭头。

## 5. 展示样式控制

说：

```text
画一个粗的紫色线条
```

说明系统能识别颜色和线条粗细。

## 6. 展示编辑能力

说：

```text
向右移动最后一个图形
```

再说：

```text
放大最后一个图形
```

说明系统支持对最近生成元素进行语音编辑。

## 7. 展示容错与修正

说：

```text
撤销
```

再说：

```text
重做
```

说明绘图过程可以通过语音修正。

## 8. 展示导出

说：

```text
导出
```

说明作品可保存为 PNG。

## 9. 设计总结

本项目将语音识别、指令解析、绘图状态和 Canvas 渲染拆成独立模块。常见指令在浏览器本地解析，不依赖每次调用云端大模型，因此响应更快，也能控制运营成本。未来可以在本地解析失败时再调用大模型，让工具同时具备稳定性和更强泛化能力。
